# My Bug-Hunting Notes

This repository is my personal knowledge base and learning journal throughout my journey in Web Application Penetration Testing and Bug Bounty Hunting.

I continuously update it as I learn, solve labs, practice on real-world scenarios, and gain experience.

Rather than being a collection of payloads or copied cheat sheets, these notes document:

* The concepts I truly understood.
* My thought process while analyzing and testing applications.
* Lessons learned from labs and practical exercises.
* Common patterns and recurring mistakes.
* Practical testing methodologies and checklists.
* Observations that I want to remember in future engagements.

The goal of this repository is to build a practical reference that grows with my experience—not just a list of notes, but a record of how my mindset evolves as a Web Security researcher.
